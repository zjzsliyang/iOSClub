#### 获取所有学校信息

```
GET /club/info/getAll
```



#### 获取所有老师

```
GET /club/teacher/getAll
```

#### 获取某个学校所有老师

```
GET /club/teacher/getByCode?code=10247
```



#### 获取所有学生干部

```
GET /club/sm/getAll
```

#### 获取某个学校所有学生干部

```
GET /club/sm/getByCode?code=10247
```
